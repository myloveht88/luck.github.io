脑图转测试用例

2023-08-31

vue2在线脑图

工作中，测试人员都习惯用xmind来写测试思路，但是项目需求和设计往往很大而杂，导致测试的xmind条目繁多，再基于xmind写成excel的用例更是费时费力。  

一直都想怎么利用开源的脑图在线工具，把大家的脑图都放到线上来，一来好维护，二来更好分享出去。在测试用例输出上做到省时省力。

找来找去发现了一个不错的开源工具：https://github.com/wanglin2

github地址：https://github.com/wanglin2/mind-map

在线预览：思绪思维导图

还有一个vue3版本：https://github.com/huangyuanyin/hyy-vue3-mindMap
